# List Practice Tasks (C)
## Task 1:
1. The function takes a list and a number, finds the first student with a grade point average less than that number and removes it.
2. The function searches for a student by name and moves it to the end of the list.
3. The function searches for a student by name and removes it.
4. The function finds the first student with a grade point average less than that number and moves it to the end of the list.

**FILE:** `single_list1.c`

## Task 2:
1. Create a list and add a node to the end and the beginning.
2. Search for an element and remove it (by value).
3. Expand the list.

**FILE:** `single_list2.c`
