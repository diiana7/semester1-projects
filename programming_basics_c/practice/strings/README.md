# String Practice Tasks (C)


## Task 1:
Create functions that: 
1.count the number of vowels in a string. 
2.remove all digits from a string. 
3.check if a word is a palindrome. 
4.replace all spaces with the _ symbol.

**File:** `string1.c`


## Task 2:
Write a function that finds and replaces all '$' characters in a string with '*'. The function
returns how many times the replacement was performed and in which positions the replacement did not occur.

**File:** `string2.c`
